# FiapRide

Esse projeto implementa conceitos de orientação a objetos em Java.

## Classes
Implementamos a classe Passageiro que controla o saldo (adição e pagamento);
São realizadas validações para que operações inválidas não sejam executadas.

Implementamos a classe Animal que controla os atributos (nome, numeroPatas e voa);
Ela implementa atributos que permitem renomear o animal e caso consiga fazê-lo voar.
